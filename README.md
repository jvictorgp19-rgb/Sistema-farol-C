# 🚨 Projeto Farol — Sistema de Segurança Pública

> Nota 10 · Abril/2026 · UNIBRA

O **Farol** é um sistema desenvolvido em **Linguagem C** focado no registro e monitoramento de ocorrências de segurança pública. Desenvolvido como requisito acadêmico para o curso de Análise e Desenvolvimento de Sistemas (ADS) no Centro Universitário Brasileiro (UNIBRA), o projeto recebeu nota máxima em código, documentação e apresentação.

---

## 🚀 Funcionalidades

- **Cadastro e Login Seguro** — Validação de CPF único e ocultação de senha durante a autenticação.
- **Registro de Ocorrências** — Cadastro detalhado com descrição, bairro, data/hora e coordenadas (latitude/longitude).
- **Feed de Ocorrências** — Visualização de todos os incidentes reportados no sistema.
- **Busca por Bairro** — Filtro para localizar incidentes em áreas específicas.
- **Mapa de Ocorrências** — Interface via terminal para marcação de zonas de risco por proximidade.
- **SOS Emergência** — Acesso rápido aos números das autoridades (190, 192, 193).
- **Painel Administrativo** — Gestão de usuários e moderação do sistema.

## ⚙️ Como Executar

**1. Compile o código com GCC:**
```bash
gcc projeto_farol.c -o farol
```

**2. Execute o programa:**
```bash
./farol
```

> Também é possível compilar e executar pelo **Dev-C++** abrindo o arquivo `.c` diretamente.
